# Walytis
Here lives the source code for Walytis, a flexible, lightweight non-linear blockchain.

This project is built to be run by [Brenthy](https://github.com/emendir/BrenthyAndWalytis/blob/master/Documentation/Brenthy/Meaning/IntroductionToBrenthy.md), a framework for developing and deploying new kinds of blockchains.
They both depend on each other, so they share the same repository.
